Landing standalone para `niuro.io/wellness`.

**Evento:** Sábado 25 · abril · 2026 · La Reina, Santiago
**Contacto:** marketing@niuro.io

## Cómo previsualizar

```bash
cd landing
python3 -m http.server 8765
# abrir http://localhost:8765
```

## Stack

- HTML + CSS + JS inline (standalone)
- HubSpot form embed (portal 50413996)
- Fonts: Host Grotesk (italic) + JetBrains Mono

Compartir `index.html` + `assets/` al equipo dev para hostear en `niuro.io/wellness`.
